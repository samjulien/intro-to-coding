# Awesome Students
Make your first PR here! I'll be going over this in the second lesson.

## Instructions
1. Sign up for [GitHub](https://www.github.com)
2. Download and install Git (instructions [here](https://gist.github.com/derhuerst/1b15ff4652a867391f03)).
3. Download and install [Visual Studio Code](https://code.visualstudio.com/).
4. Fork this repo.
5. Clone your new fork and open it in VS Code: 

```bash
git clone https://github.com/you/intro-to-code.git
cd intro-to-code
code .
```

6. Add yourself to the list below.
7. Commit and push up your changes.
8. Submit your first pull request.
